# P1 / P2 入门实战：切块 + 向量语义检索

> **P1**：把 `knowledge/*.md` 切成多段，存入 `data/chunks.json`  
> **P2**：为每段算 Embedding，检索时用语义相似度（不再只靠关键词）

前置：已完成 [P0入门实战.md](P0入门实战.md)。

---

## 你要先懂的两句话

- **P1**：一篇体育馆 md 可以拆成「场地」「预约」多段，RAG 只把**相关段**送给模型。  
- **P2**：用户说「打羽球」，即使正文写的是「羽毛球」，向量检索也能拉近。

---

## 第 1 步：更新 config.py

若你的 `config.py` 是 P0 时复制的，请对照 `config.example.py` **补上**这些项（Key 不用改）：

```python
DATA_DIR = "data"
CHUNKS_FILE = "chunks.json"
EMBEDDINGS_FILE = "embeddings.npz"
CHUNK_MIN_CHARS = 30
BUILD_EMBEDDINGS_ON_INDEX = True
EMBEDDING_MODEL = "paraphrase-multilingual-MiniLM-L12-v2"
USE_VECTOR_INDEX = True
SIMILARITY_THRESHOLD = 0.35
```

## 第 3 步：让知识库够「15 个文档」

任务书里的 **≥15 段资料**，在本项目中落实为：`knowledge/` 文件夹里 **≥15 个**独立的场馆 `.md` 文件（每个馆一份，类似 `示例-体育馆.md`）。

| 计数对象 | 要求 | 说明 |
|----------|------|------|
| **`.md` 文件个数** | **≥15** | 任务书硬性要求 |
| **`##` 切块段数** | 越多越好 | 仅用于 RAG 检索，**不是**「15 段」的含义 |

当前示例只有 2 个文件，还需再添加 **至少 13 个**本校真实场馆文档。

`indexer.py` 会把每个文件再按 **`##` 二级标题** 切成多段写入 `chunks.json`（与「15 个文件」是两层概念）。

**自检**：运行 `python test_retriever.py` 会显示「场馆文档：N 个」。

### 期望输出

```text
[indexer] 知识库文档 2 个 .md，切块后 6 段 → data\chunks.json
[indexer] 提示：任务书要求 ≥15 个场馆文档，当前 2 个，请继续在 knowledge/ 添加 .md
```

- 报错 `No module named sentence_transformers`：重做第 2 步。  
- 只出现「已保存 N 个段落」但向量失败：见下文 **模型下载失败**。

**以后每次改 knowledge 都要再跑一次** `--rebuild`。

### 模型下载失败（WinError 10060 / Can't load the model）

**先分清：**

| 现象 | 含义 |
|------|------|
| `已保存 N 个段落 → chunks.json` | **P1 已成功**，可继续 |
| 后面一长串 `huggingface.co` 超时 | **P2 下载向量模型**失败，不是 P1 坏了 |
| 你按了 Ctrl+C | 会显示 `KeyboardInterrupt`，同样发生在 P2 下载阶段 |

**处理步骤（按顺序试）：**

1. 确认 `config.py` 里有 `HF_ENDPOINT = "https://hf-mirror.com"`（新版 `embedder.py` 会自动用）。  
2. **删掉半拉子缓存**（在 PowerShell 执行）后重试：

```powershell
Remove-Item -Recurse -Force "$env:USERPROFILE\.cache\huggingface\hub\models--sentence-transformers--paraphrase-multilingual-MiniLM-L12-v2" -ErrorAction SilentlyContinue
python indexer.py --rebuild
```

3. 仍失败则 **先跳过 P2**，只做 P1：

```powershell
python indexer.py --rebuild --no-embed
```

并在 `config.py` 设 `USE_VECTOR_INDEX = False`，照样可 `python test_retriever.py`（对 chunks 做关键词检索）。

4. 有 VPN 时关掉镜像直连，或换手机热点再 `--rebuild`。

## 数据流（P1/P2 之后）

```text
knowledge/*.md
    → indexer.py --rebuild
    → data/chunks.json + data/embeddings.npz
用户问题
    → retriever.py（向量相似度 Top-K）
    → main.py 拼 Prompt
    → llm.py（DeepSeek）
```
